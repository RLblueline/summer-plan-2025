Thu-Fri: Sensor Integration

HC-SR04 Ultrasonic sensor integration. Make servo track your hand distance.

The Pro Tip

Always use a separate power supply for motors (e.g., a 4xAA battery pack). Drawing motor current through an Arduino or Pi is the #1 way to kill your board on Day 3.

Week 2: Engineering Python & Math

JUNE 21 - JUNE 27

The Syllabus

  • NumPy: Matrices are everything
  • OOP: Class-based Robot control
  • Calculus: The logic of Gradients

Milestone

By Friday, you should be able to write a script that calculates a 3D coordinate from three joint angles using a rotation matrix.

Math Secret

Don't memorize formulas. Understand Coordinate Frames. If you can track where a frame is relative to another, you can control any robot.
